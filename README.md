# JoyMan - 分布式安卓任务管理系统

## 📱 项目简介
JoyMan 是一款完全本地运行的分布式安卓任务管理系统，与未来姐姐 AI 助手深度集成。支持设备间 P2P 同步，无需依赖云端服务。

## ✨ 核心特性
- **完全本地运行**: SQLite 数据库，数据完全掌控在自己手中
- **Redmine 兼容 API**: 最小化迁移成本，支持从 Redmine 导入任务
- **分布式 P2P 同步**: 设备间通过 WiFi/蓝牙分享任务包
- **随机数字 ID**: 避免 ID 冲突，支持离线生成
- **冲突容忍机制**: 导入时自动检测并重新分配冲突 ID
- **AI 语音集成**: 通过未来姐姐语音创建/查询任务

## 🏗️ 技术架构
- **平台**: Android (纯 Java)
- **数据库**: SQLite
- **ID 生成**: 时间戳 + 随机数（约 14 位数字）
- **同步协议**: JSON 任务包 + 数字签名
- **冲突处理**: 基于标题 + 创建时间的唯一性检测
